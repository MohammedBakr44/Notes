# 一段動詞
## -いる・-える
*-い means steam, basically any letter ends with い e.g: き、し、み*
### て：て、た
例：
食べる -> 食べて

# 五段動詞
## う、つ、る
### て: って｜った
例:
笑う->笑って

持つ -> 持って
## ぬ、ぶ、む
### て:んで｜んだ
例：
死ぬ -> 死んで

飲む -> 飲んで
## く、ぐ
### て：いて｜いた、いで｜いだ
例：
歩く-> 歩いて

泳ぐ-> 泳いで
## す
### て：して｜した
例：
話す -> 話して
